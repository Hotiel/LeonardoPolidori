  • Web: "La página invisible"

    "La página invisible" (JUL 2025) is a fictional town bookstore, specialized in fantasy novels and known for organizing medieval-style fairs. I created a website for it featuring an introduction, monthly book recommendations, contact section, and custom design. This project marked my first time working with icon libraries, pre-designed color palettes, and nav bars. Although it still lacks responsive design, it’s the direct ancestor of this current portfolio.

  • Web (CYOA game) "La mujer de rojo"

    "La mujer de rojo" (JUN 2025) was my first web design. It’s a site where I experimented with internal page linking and file referencing. What stands out the most, however, is the site’s content — a short story that allows a closer look at my creative side. In hindsight, I can clearly spot several design flaws, more so than in my later work. Still, I see it as a solid starting point — and an interesting project to revisit in the future.

  • Análisis y normalización de base de datos

    (MAY 2025) For this project, focused on data analysis, normalization, and presentation, I worked with a public dataset of Amazon products. In the GitHub repository and the included README file, you’ll find details on the decisions made throughout the project. The visualizations were created using Matplotlib and Seaborn, and the environment included SQLite for SQL queries and VSCode for Python development.
